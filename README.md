# Buntykapila Photography Website

A responsive one-page cinematic photography website.

## Open locally
Open `index.html` in a modern browser.

## Deploy
Upload the entire folder to Netlify, Vercel, GitHub Pages, Cloudflare Pages, cPanel or any static website host.

## Important
- Contact form opens WhatsApp and does not store data.
- Pricing is shown in page text, not embedded in portfolio photos.
- Wedding coverage is presented as a new service; the site does not claim an existing wedding portfolio.
- Add verified client reviews later rather than publishing invented testimonials.
